   - Use `useMemo` to compute layout: `hierarchy({ name: 'root', children: items })` -> `.sum(d => 'value' in d ? toChartValue(d.value) : 0)` -> `.sort((a, b) => (b.value ?? 0) - (a.value ?? 0))`. Then `treemap().size([width, height]).padding(3).round(true).tile(treemapSquarify)`.
   - State: `hoveredItem: string | null` for hover highlighting.

   - Hover: `onMouseEnter`/`onMouseLeave` set `hoveredItem`. Non-hovered items get `opacity={0.5}` with CSS `transition-opacity duration-150`.
   - Text: white fill, `pointerEvents: 'none'`, bold name, lighter opacity for dollar amount.
   - Import: `{ hierarchy, treemap, treemapSquarify } from 'd3-hierarchy'`, `{ toChartValue } from '@/lib/chart-utils'`, `{ formatDollarsAbbreviated } from '@/lib/format'`, `{ useRouter } from 'next/navigation'`.
