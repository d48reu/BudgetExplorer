Create `src/components/charts/YearOverYearChart.tsx` as a Client Component:

- `'use client'` directive
- Import `useMemo` from React, `scaleBand` and `scaleLinear` from `d3-scale`, `toChartValue` and `formatPercentage` from chart-utils, `formatDollarsAbbreviated` and `formatYoYChange` from format, `ChartContainer`, `DataTableToggle`, `TableColumn` and `SerializedYoYData` from types/budget

**Props type:**
```typescript
type YearOverYearChartProps = {
  data: SerializedYoYData[]
  areaColor: string
}
```

**Inner chart component** receives `{ width, height }` from ChartContainer render prop:
- Margins: top=30, right=20, bottom=40, left=80
- `xScale = scaleBand().domain(data.map(d => d.fiscalYear)).range([0, innerWidth]).padding(0.3)`
- `yScale = scaleLinear().domain([0, max totalBudget]).nice().range([innerHeight, 0])`
- Render SVG with role="img" and aria-label="Year-over-year budget chart"

**Bars:**
- For each data item, render a vertical `<rect>`:
  - x = xScale(d.fiscalYear)
  - y = yScale(toChartValue(d.totalBudget))
  - width = xScale.bandwidth()
  - height = innerHeight - yScale(toChartValue(d.totalBudget))
  - fill = d.isCurrent ? areaColor : '#D1D5DB' (neutral gray per CONTEXT.md)
  - rx = 2

**X-axis labels:**
- For each data item, render `<text>` below bars:
  - x = center of bar (xScale(d.fiscalYear) + bandwidth/2)
  - y = innerHeight + 24
  - textAnchor="middle", className "fill-text-secondary text-xs"
  - Content: fiscal year label (e.g., "FY 2025-26")

**Y-axis labels:**
- Compute ~4-5 tick values from yScale.ticks(4)
- For each tick, render `<text>` to the left:
  - x = -12, y = yScale(tick), textAnchor="end"
  - Content: formatDollarsAbbreviated(tick * 100) -- convert back to cents for the formatter
- Render horizontal grid lines at each tick: `<line>` with stroke="#E5E7EB" strokeDasharray="4 4"

**Value labels on bars:**
- For each bar, render formatted dollar amount above the bar:
  - x = center of bar
  - y = yScale(toChartValue(d.totalBudget)) - 8
  - textAnchor="middle", className "fill-text-primary text-xs font-medium"
  - Content: formatDollarsAbbreviated(d.totalBudget)

**Percentage change badge (current year only):**
- Find the current year entry (isCurrent === true)
- Find the immediately prior year entry (the one before current in the sorted data)
- If both exist, compute change using formatYoYChange(current.totalBudget, prior.totalBudget)
- Render a badge above the current year's value label:
  - Small `<rect>` background (rounded) with fill based on direction: blue (#0057B8) for increase, orange (#F7941D) for decrease, gray for unchanged
  - `<text>` inside with white fill showing the percentage (e.g., "+5.2%")
  - Position: centered above current bar, y offset above the value label

**Data availability note:**
- If data.length < 5, render below the chart (outside SVG): a text note "Data available from {earliest fiscal year}" in text-xs text-text-tertiary

**DataTableToggle wrapper:**
- Define table columns: Fiscal Year (left), Total Budget (right, formatDollarsAbbreviated), Operating (right, formatDollarsAbbreviated), Capital (right, formatDollarsAbbreviated)
- Wrap in DataTableToggle with chartLabel="Year-over-year budget chart" and tableLabel="Year-over-year budget data"

**ChartContainer wrapper:**
- minHeight = 320
- ariaLabel = "Year-over-year budget comparison"

**Edge cases:**
- If data has only 1 entry, show single bar with no percentage badge
- If data is empty, don't render anything (parent handles this)

Export as named export `YearOverYearChart`.
