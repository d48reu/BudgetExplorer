       - Explain: for fiscal years not covered by the current Budget in Brief PDF, create CSV files named fy_YYYY_YY_departments.csv (e.g., fy_2021_22_departments.csv)
       - Explain the department_alias column: if a department was renamed (e.g., "Miami-Dade Police Department" became "Sheriff" in 2025), put the old name in department_name and the current name in department_alias
       - Note that operating_budget and capital_budget are in whole dollars, not cents

    3. Create `pipeline/transform/historical.py`:
       - parse_historical_csv(csv_path: str) -> list[dict]:
         * Read CSV file using Python csv module (not pandas -- keep dependencies minimal)
         * Validate required columns exist
         * For each row: convert operating_budget and capital_budget from dollar strings to cents via dollars_to_cents
         * Handle department_alias: if present, use it to look up the current department; if blank, use department_name
         * Return list of dicts ready for seeding

       - parse_historical_json(json_path: str) -> list[dict]:
         * Alternative JSON format support
         * Same structure as CSV but in JSON array format
         * Convert dollar amounts to cents

       - detect_format(file_path: str) -> str:
         * Returns "csv" or "json" based on file extension

    4. Create `pipeline/load/seed_historical.py`:
       - resolve_department(conn, name: str, alias: str | None) -> int | None:
         * First try exact match on departments.name or departments.slug
         * If alias provided, try matching alias against departments.name
         * If still no match, check department_aliases.historical_name
         * Return department_id or None (with warning log)

       - seed_historical_year(conn, fiscal_year_label: str, data: list[dict]):
         * Create or get fiscal_year record (derive start_date/end_date from label pattern "FY YYYY-YY")
         * Delete existing department_budgets for this fiscal year (idempotent)
         * For each record: resolve department via resolve_department(), insert into department_budgets
         * Log summary: N departments seeded, N skipped (unresolved)

       - seed_all_historical(conn, data_dir: str):
         * Scan data_dir for CSV/JSON files matching fy_*_departments.* pattern
         * For each file: parse and seed
         * Print summary of all years seeded

    5. Update `pipeline/cli.py`:
       - Add `seed-historical` command:
         * --data-dir option (default: pipeline/data/historical/)
         * Calls seed_all_historical()
         * Prints summary of fiscal years and departments seeded
       - Add `seed-historical-file` command:
         * --file option for a single CSV/JSON file
         * --fiscal-year option (required)
         * Calls parse + seed for one file
