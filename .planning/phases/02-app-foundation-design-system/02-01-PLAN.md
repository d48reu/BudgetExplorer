**Step 1: Prisma introspection**

Initialize Prisma and introspect the existing database:
```bash
cd budget-explorer-web
pnpm prisma init
```

Set `DATABASE_URL` in `.env.local` (or `.env` inside budget-explorer-web) to point at the same PostgreSQL database used by the pipeline. Then:
```bash
pnpm prisma db pull
pnpm prisma generate
```

This generates `prisma/schema.prisma` with models for all tables: fiscal_years, strategic_areas, departments, department_budgets, strategic_area_budgets, revenue_by_source, millage_rates, etc. BigInt columns will map to Prisma's `BigInt` type automatically.

**Step 2: Prisma singleton client**

Create `src/lib/prisma.ts` using the global singleton pattern to prevent connection pool exhaustion during hot module reloading in development. Pattern from 02-RESEARCH.md:
```typescript
import { PrismaClient } from '@prisma/client'
const globalForPrisma = globalThis as unknown as { prisma: PrismaClient }
const prisma = globalForPrisma.prisma || new PrismaClient()
if (process.env.NODE_ENV !== 'production') globalForPrisma.prisma = prisma
export default prisma
```

**Step 3: Data access layer with BigInt serialization**

Create `src/lib/db/queries.ts` with these exported functions:
- `getCurrentFiscalYear()` — returns the FY 2025-26 record with all BigInt fields converted to strings
- `getStrategicAreas()` — returns all 9 strategic areas with their budget data for FY 2025-26
- `getDepartmentCount()` — returns count of departments (35)
- `getQuickStats()` — returns aggregated stats for the homepage (strategic area count, department count, total employees, fiscal year label)

Every function MUST convert all BigInt values to strings before returning. Never pass raw Prisma objects to components. Use the `SerializedFiscalYear`, `SerializedStrategicArea` types.

**Step 4: TypeScript types**

Create `src/types/budget.ts` with serialized types:
- `SerializedFiscalYear` (id, label, totalOperating: string, totalCapital: string, totalBudget: string, totalEmployees)
- `SerializedStrategicArea` (id, name, slug, color, operatingBudget: string, centsPerDollar)
- `QuickStats` (strategicAreaCount, departmentCount, totalEmployees, fiscalYear)

**Step 5: Dollar formatting utilities**

Create `src/lib/format.ts` with:
- `formatDollarsAbbreviated(cents)` — cents string to "$13.2B", "$857.6M", "$1.2K"
- `formatDollarsFull(cents)` — cents string to "$13,233,238,000"
- `formatYoYChange(currentCents, priorCents)` — returns `{ value: "+5.2%", direction: "increase"|"decrease"|"unchanged" }`

Implementation per 02-RESEARCH.md Pattern 3.
