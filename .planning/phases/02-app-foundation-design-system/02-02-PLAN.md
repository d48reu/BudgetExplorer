**Step 1: Tailwind v4 design tokens in globals.css**

Replace `src/app/globals.css` with the full design system. Use the `@theme` directive (Tailwind v4 CSS-first config — NOT tailwind.config.js). Include:

```css
@import "tailwindcss";

@theme {
  /* Miami-Dade brand colors (for data viz, not UI chrome) */
  --color-mdc-blue: #0057B8;
  --color-mdc-orange: #F7941D;
  --color-mdc-green: #00A651;
  --color-mdc-red: #EF4444;

  /* Neutral UI palette */
  --color-surface: #FFFFFF;
  --color-surface-secondary: #F9FAFB;
  --color-border: #E5E7EB;
  --color-border-strong: #D1D5DB;
  --color-text-primary: #111827;
  --color-text-secondary: #6B7280;
  --color-text-muted: #9CA3AF;

  /* YoY change colors (neutral, not good/bad — per user decision) */
  --color-change-increase: #0057B8;
  --color-change-decrease: #F7941D;

  /* Font families */
  --font-heading: var(--font-inter), sans-serif;
  --font-body: system-ui, -apple-system, "Segoe UI", Roboto, sans-serif;

  /* Responsive breakpoints */
  --breakpoint-xs: 23.4375rem;  /* 375px */
  --breakpoint-sm: 40rem;       /* 640px */
  --breakpoint-md: 48rem;       /* 768px */
  --breakpoint-lg: 64rem;       /* 1024px */
  --breakpoint-xl: 80rem;       /* 1280px */

  /* Spacing scale for dashboard density */
  --spacing-card: 1.25rem;      /* 20px */
  --spacing-section: 2rem;      /* 32px */

  /* Z-index scale (prevents stacking conflicts) */
  --z-base: 0;
  --z-nav: 40;
  --z-tooltip: 50;
  --z-modal: 60;
}
```

Also add base body styles and utility classes for the card and tooltip aesthetics.

**Step 5: BudgetTerm tooltip component**

Create `src/components/ui/BudgetTerm.tsx` using Floating UI:
- `'use client'` directive (interactive component)
- Props: `term: string`, `definition: string`, `children: React.ReactNode`
- Wraps children in `<span>` with `border-b border-dotted border-text-secondary cursor-help` styling (dotted underline per user decision)
- Uses `useFloating` with `offset(8)`, `flip()`, `shift()` middleware
- Uses `useHover`, `useFocus`, `useDismiss`, `useRole` interactions
- Tooltip renders via `FloatingPortal` with z-index at `z-tooltip` level
- Shows term name in bold and definition text
- Accessible: `tabIndex={0}`, `aria-describedby`, `role="tooltip"`

Implementation matches 02-RESEARCH.md Pattern 5 exactly.
